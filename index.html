Input Cheatsheet


Input Tags

Tag Attributes Description
<form> action, method main form element
<input> type, id, name, required, min, max, value, pattern allows for input
<label> for, form labels the input
<textarea> name, form, wrap, rows, maxlength, required an area for larger chunks of input
<select> name, form, multiple, required, autofocus drop down list
<datalist> id set of items from dropdown menu
<option> label, selected, value allows adding option to list
